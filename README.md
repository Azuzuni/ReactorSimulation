# ReactorSimulation
Goal of this project is to visualize what happens inside nuclear reactor
inspired by:
  https://www.youtube.com/watch?v=P3oKNE72EzU&t=584s


# Dependencies
- CMake 3.20 or higher
- Raylib (downloaded automatically if ran with preset raylib-entt)
- SFML (downloaded automatically if ran with preset sfml-entt)
- EnTT (downloaded automatically if ran with preset raylib-entt or sfml-entt)

# Controls
UP and DOWN keys to change position of control rods
